# Таблица криптовалюты.

### Как начать

Для того, чтобы начать, используйте команду `npm install` в терминале. Она установит все нужные зависимости.
После установки, используйте команду `npm start`, она развернет локально приложение и откроет его в браузере.
ВАЖНО: приложение готово только в режиме разработчика, команда `npm run build` работать корректно не будет.

### Описание

Небольшое приложение, которое отображает данные о криптовалюте в реальном времени.
При маунте компонента таблицы, делается запрос на сервер. После отрисовываются данные, и открывается веб-сокет, который позволяет постоянно обновлять цену отоброжаемой валюты.
Использованный [api](https://docs.coincap.io).
